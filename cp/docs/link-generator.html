Link Generator
Table of Contents
Overview
The link generator can be used to assist you in generating the out.php links you will use on your site to skim traffic, send traffic to trades, etc.
Link Generator Settings
    Link Type - Select the type of link to generate.

    % to Content
    This link type will include a hard-coded skim percentage in the URL to determine when surfers are sent to trades.

    Skim Scheme
    This link type will use the skim scheme that you select to determine when surfers are sent to trades.

    To Specific Trade
    This link type will always send the surfer to the trade listed in the URL.


    % to Content - Enter the percentage (a value from 1 - 100) of clicks on this link that will be sent to content. The remaining percentage will be sent to trades.
    First click always to content
    Check this if you always want the first click on this type of link to go to the content URL
    Skim Scheme - Select the skim scheme to use for this link
    Trade - Select the trade this link should send to
    Content URL - Enter the full URL to the content this link should send to when it does NOT send to a trade
    URL Encoding - Select how the content URL should be encoded.

    urlencode
    The content URL will be visible in the link, but special characters will be encoded so they can be passed through the out.php script without a problem

    base64_encode
    The content URL will be completely encoded and not readable in the link.
    Limit to Category - Select a category if you want to limit this link to send only to trades in that category
    Limit to Group - Select a group if you want to limit this link to send only to trades in that group
    Link Identifier - An identifier for the generated link. This can be used to break down stats further by what link was clicked.
