Blacklist
Table of Contents
Overview
The blacklist feature allows you to ban certain items from your trade accounts. The blacklist is checked for all new trade registrations, so you can prevent known cheaters or other trades that you have had issues with from registering new trade accounts at your site. There are several different blacklists available, and each is described below.

E-mail Addresses - Use this blacklist to ban certain e-mail addresses from registering new trade accounts

Domains - Use this blacklist to ban domains from registering new trade accounts

User IP - Use this blacklist to ban IP addresses of end-users from registering new trade accounts

Server IP - Use this blacklist to ban the IP addresses of servers where a trade site is hosted from registering new trade accounts

Nameservers - Use this blacklist to ban domains using certain nameservers from registering new trade accounts

Words - Use this blacklist to ban words from the site name or description

Page Content - Use this blacklist to ban any HTML code or text from appearing in the trade's traffic URL page

HTTP Headers - Use this blacklist to ban values contained in the HTTP headers of the trade's traffic URL
